# Nasazení verze 1.0

1. Nahrajte obsah ZIP do repozitáře.
2. Ověřte, že se v hlavičce zobrazuje `Business Risk Hub 1.0 · Core Stabilization`.
3. Načtěte uloženou DB poptávku.
4. Zkontrolujte, že počet nabídek odpovídá konkrétní poptávce.
5. V Nabídkách upravte jednu nabídku a uložte.
6. Ověřte Porovnání a Zprávu.

Doporučení: po nasazení v prohlížeči použít tvrdý refresh.
