# Business Risk Hub 3.0.0 – Case Workspace Rebuild

Tato verze přestavuje pracovní logiku aplikace na jeden obchodní případ jako jediný zdroj pravdy.

## Hlavní změny

- odstranění starých vývojových bloků z dashboardu,
- sjednocení práce do záložek obchodního případu,
- karta klienta je kořen případu,
- pojišťovny jsou součástí konkrétního případu,
- nabídky se vyplňují v jedné tabulce podle rizik,
- porovnání vychází z těchto nabídek,
- doporučení potvrzuje poradce,
- klientský výstup bere data z aktivního případu.

## Zachováno

- databáze,
- API pro poptávky,
- Admin,
- dokumentová knihovna,
- textace,
- rizikový model,
- ASTORIE branding.
