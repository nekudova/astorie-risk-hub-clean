# Business Risk Hub 3.0.3

Stabilizační oprava pracovní architektury po 3.0.2.

## Opraveno
- Karta klienta je oddělena od odborné karty pro pojištění.
- Karta pro pojištění obsahuje provozní, ekonomické, majetkové, odpovědnostní a podkladové údaje.
- Admin je znovu rozšířen jako provozní číselník.
- Textace se zobrazují v samostatné sekci a spravují v Adminu.
- Databáze se nemění destruktivně; ukládá se do existujícího full_payload a katalogových nastavení.
